## How I Think About AI

I don't just use AI as a tool — I study how it behaves. I care about:

- **Reliability** — how systems fail, not just how they succeed
- **Decision boundaries** — where models should and shouldn't have autonomy
- **Safety** — building AI that behaves as intended under pressure
- **Real-world deployment** — the gap between demo and production

These aren't abstract interests. They come from building Pilot and watching what happens when an AI agent has to make real decisions in real time.
